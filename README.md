# MY NEWS APP :red_envelope:

<br>

### A simple news app for you to scroll through all the relevant headlines and be up-to-date. :grinning:

## The Technologies and Tools used while making this application are:  :rocket: :metro:
                    
                     * Android Studio
                     * RESTful API's
                     * Picasso
                     * PostMan
                     * GSON Library
